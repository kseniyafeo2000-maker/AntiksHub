# 🚀 AntiksHub - Маркетплейс на Django

Современный маркетплейс с оранжевой темой для продавцов и покупателей.

## 🎯 Особенности
- Оранжевый дизайн 2026 года
- База данных продавцов и покупателей
- Каталог товаров с изображениями
- Корзина покупок
- Система заказов
- Админ-панель Django

## 🛠️ Технологии
- Python 3.12
- Django 6.0
- SQLite3 (разработка)
- HTML/CSS/JavaScript
- Pillow (изображения)
